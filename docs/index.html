Your loop can live right in the registry (no repo needed) or in your own module. Either way, three commands ship it to this page:

1 · scaffold
karmax loops new my-loop
# loop.go + loop.json + README
2 · implement
k.Notify, k.Harness, k.Remember,
k.Propose, k.Remind, k.HTTP …
3 · publish
karmax loops publish my-loop
# validates, compiles, opens the PR
