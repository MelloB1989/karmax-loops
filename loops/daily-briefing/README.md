# daily-briefing

**Default loop.** At 08:30 the agent compiles your morning briefing — today's calendar
and reminders, the latest tech-news digest from memory, open coding sessions, anything
pending or urgent from WhatsApp — and the loop delivers it deterministically: always to
the phone app feed (+push), and optionally to WhatsApp.

## Config

- `KARMAX_LOOP_DAILY_BRIEFING_WHATSAPP` — a number/chat to also deliver the briefing to.
